# SmartPhish

SmartPhish is a GAN-powered phishing URL detection system designed to identify zero-day phishing attacks using adversarial training and lightweight neural networks. The system provides real-time protection using a browser extension.

---

## 🔐 Key Highlights

- Zero-day phishing detection
- GAN adversarial training
- Lightweight neural network
- Real-time Chrome extension
- Low latency prediction
- No DNS or third-party API dependency

---

## 🧠 Tech Stack

- Python
- TensorFlow / Keras
- Flask / FastAPI
- NumPy, Pandas, Scikit-learn
- HTML, CSS, JavaScript
- Chrome Extension (Manifest V3)

## ⚙️ Installation

### Backend Setup

```bash
pip install -r requirements.txt
python app.py
```

### Chrome Extension Setup
```
1. Open Chrome
2. Go to chrome://extensions
3. Enable Developer Mode
4. Click Load Unpacked
5. Select extension folder
```
